StageFit — plateforme Angular 18 (standalone components) de gestion d’offres de stage avec filtres avancés, favoris, assistant IA de recommandation et module de candidature (upload liens CV/portfolio, suivi d’état).
Frontend : Angular 18, TypeScript, RxJS, routing, SSR-ready, formulaires réactifs, export CSV, UI responsive.
Backend mock : JSON Server (API REST simulée) + persistance locale (favoris, auth mock).
Authentification par rôles (recruteur/candidat) : ajout/édition réservé aux recruteurs, consultation/postulation pour candidats.
Mise en avant du matching : assistant IA calcule score et probabilité d’acceptation selon compétences/langues/mode/localisation/expérience.
Valeur ajoutée : accélère le tri et la sélection des candidats,propose des créneaux d’entretien (lien calendrier) et un parcours candidat fluide.
